The code epidemic

The Origin

We want to bring a transition in how we learn and ad interim implicate a learning by doing philosophy amongst students who are close to computers. Hence, A platform namely, a computer society/club which solely addresses to help students write code and become programmers. And open to anyone with a knack of learning computers.

The Host

Now the question is how?

In order to interact with a machine, one must speak its language. And one merely does not picks up a language in a day or two, its done over a course of time. And even then you realize that its not enough.

Hence, we achieve such a task only by learning and doing incessantly.

The Medium

To embrace continuous learning we will be conducting and observing all of the events listed below

  • - Tutorials
  • - Workshops
  • - Code week
  • - Competitions
  • - Meetups
  • - Hackathon

The Spread

Everyone is a teacher and everyone is a student. The motive here is that, anyone can learn anything about computers. What we want to implement is a free environment with no boundaries to learning.

Every week we will decide what to learn, and that idea can come from anyone. For example, if a person wants to learn how to write javascript and another one wants to understand working of GNU/Linux, and yet another wants to write his/her own compiler, we’ll incorporate everything. Thus we study/learn what everyone wants to learn but don’t get a chance to.

We have modelled this approach as not to just write code but also to understand a computer inside out, as a whole entity. Thus, it starts from learning how the computer works to how one can program a computer to do their work.

The Cause

To become a programmer one must have an insight of how stuff works and that is achieved by inducing

  • - Better programming habits
  • - Importance of a text editor
  • - Using tools
  • - Writing production code
  • - Learning new languages
  • - Working on different operating system
  • - Writing more and more code

The Confinement

A holistic learning environment must have everything within proximity. Hence, "Kronos" exists on the Internet.

  • - Open computers without a limited access code are crucial. As there will be many softwares and tools that will be required for use during programming. Also, it ensures that we can install GNU/Linux or any other Operating System and work on them and understand their working too.
  • - Abstention simply causes hindrance in learning process.
  • - We believe that internet is freedom since it includes everything we need - warez, compilers, code bases and more such stuff.

For example, Web development is one such domain which simply cannot be held within boundaries.Web APIs of twitter, facebook, youtube and other websites, game development, Forums, making a web crawler and scraping web pages is all part of programming. Hence, physical presence will subdue the full powers we can derive from the internet.

The Observation

A disciplined yet fun learning environment is our goal.
