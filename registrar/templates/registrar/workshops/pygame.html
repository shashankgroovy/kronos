Windows

We assume you are familiar with the general procedure of installing software on Windows. First off, download the binary(exe/msi) according to your architecture (32-bit or 64-bit). Don't know your architecture type? Head on to Microsoft's website to find out, and then come back here.

  • Python for 32-bit
  • Python-Pygame for 32-bit
  • Python for 64-bit
  • Python-Pygame for 64-bit

Now lets test off to see if everything works. Open up Python's shell and type:

  • import pygame; print pygame.version.ver

If you get something like 1.9.Xrelease (where X is any number) then Voila! everything works.

Congratulations! You're all ready for the big day.

Linux

If you are using Linux (woohoo!) then you already know what to do and how to do it, so we'll leave all the work for you :) But still if you get stuck somewhere, you can either ask the almighty Google for help or (even better) use the source! (another riddle? Great!)

. . .

So you're done, eh? Good work! Now let's install pygame. In your terminal, type this:

  • sudo apt-get install python-pygame
Note: If your os doesn't use apt, search for pygame in official repositories

When its complete, in the terminal, type this:

  • python2.7 -c "import pygame; print pygame.version.ver"

If you get something like 1.9.Xrelease (where X is any number), then congratulations, you're all set!

Mac

Check to see if you have python installed on your system, Open up your terminal and type,

  • python -c "import sys; print (sys.version)"

If it says 2.7.X then you have correct version of python and you can skip to end of this section and install pygame now. If it says something like 3.X.X or gives an error, then python's missing. Read on to know how to install it.

You can install python by downloading disk images for x64 OSX (10.6+) or x86 OSX (OSX 10.3+) and install it. If it works, you can skip to end of section. If if didn't work then, read on!

If you have brew installed, you can simply do brew install python2.7 to install python. If not, lets install brew first. Before we install python, we need GCC compiler. On OSX GCC comes with a bundle called XCode. Check to see if you have it. If not, you can get XCode from here

After you've installed XCode, you can install brew by running this script in terminal

  • ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

After that, open up your ~/.bashrc in terminal and add the following line at bottom

  • export PATH=/usr/local/bin:/usr/local/sbin:$PATH
  • export PATH=/usr/local/share/python:$PATH

We've added python's path in PATH variable before-hand to just reduce some work. Now finally lets install python.

  • brew install python

Finally, you can download pygame's disk image and install it. Phew!
