Description

A One day workshop where you will learn modern web technologies like HTML5, CSS3 and few more cool concepts like Git and Github and club all these to build your own Jekyll powered blog.

Content

  • Zero Hour :: Get work machines ready for the workshop • Includes installing Ruby, text editor, git and Making an account on Github • Details are provided below
  • Hour One :: HTML5 Rocks!
  • Hour Two :: CSS3 Rules!
  • Hour Three :: Learning Version Control : Git • Making a Github account • Igniting the Open Source fire
  • Hour Four :: Building the blog • Deploying it online

Requirements

  • Curiosity
  • A Laptop with ruby, git, jekyll installed in it. (See installation instructions below)

Don't forget this part! Setting up your systems just before the workshop is not possible. Not in this case!

Windows

We seriously recommend that you use linux, but people just won't listen. Well, what can we say, read on...

Julian Thilo has written a step-by-step tutorial on how to get ruby and jekyll on windows here. So, well leave reading it and following the instructions to you.

Note: On 3rd step, use rouge and skip pygments. Test everything and confirm that it works

When you're done, go ahead and install git. Goto msysgit.github.io and install.

. . .

Linux

If you just installed linux, conratulations! If you've been using linux for a long time then stop reading and go do the work yourself! So, for the newbies, open up the terminal and type in the following (except the $) and hit enter.

  • sudo apt-get install git ruby rubygems && sudo gem install jekyll

Note: replace apt-get with whatever package manager your OS uses.

And that's it! Now, aren't you glad that you installed linux.

. . .

Now that you're done, its time to make a github account. Go to github.com and sign up.
